Resources & Downloads

Official Anaplan Apps team collateral from the NY Connect 2026 Workshop

All materials below are distributed via the shared drive folder provided before the workshop. This page is a reference index — use it to find the right document for each phase of an implementation.

💡 How to Use These Resources

The Data Prep Workbooks should go to the customer as early as possible — ideally before the SOW is signed. The Discovery Templates are for pre-project scoping conversations. The User Stories documents help set scope and manage expectations. The Post-Generation guides are your field references during model readiness.
